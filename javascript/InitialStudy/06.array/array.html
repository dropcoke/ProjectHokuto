配列

いろんな値を引き出しの中にしまおう
ここで値を配列(引出し)に保存して、その結果をコンソールと画面に表示します。

コンソールプログラム

引出しにものをしまってその中身を確認しよう


「プログラム実行」ボタンをクリックしてください。
引き出しの中身が表示されましたか?
このプログラムでは配列に値を格納して今度はその配列の中身を繰り返して表示します。
配列には複数の値が保管できます。
[](大括弧)に
「,(カンマ)」区切りで値を設定することでそれらの値が保管されます。
配列の要素(中身の一つは)は何番目に入っているかを指定することで取り出すことができます。
javascriptをはじめ多くのプログラムでは
配列の要素は0から始まります。(1番最初に格納されたものは0番の引出しに入ります。)
上記のプログラムでは0番目から要素の数分だけ配列の中身を取り出して表示しています。
myDrawer.length のlengthは配列の要素の数になります。

DVDは何番目?


「プログラム実行」ボタンをクリックしてください。
DVDが入っている引出しを見つけられましたか?
このプログラムでは何番目の引出しに「DVD」が入っているか探しています。
配列の要素の値がDVDの場合に表示しています。
