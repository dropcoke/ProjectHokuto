プログラムで何ができるの?

みなさんパソコン、スマホでアプリを利用されていると思いますが、これらは全てプログラミング言語で書かれています。 どんなことが書かれているの? 基本的には何をどうしてくださいとコンピュータに命令をしています。 その命令をするための言語がプログラミング言語です。

    Javascript言語とは

    JavaScript はウェブページにて複雑な機能をできるようにするプログラミング言語です
    — ウェブページが読み込まれるたびに、単にあなたが見ている静的な情報を表示する以上のことをしています —
    更新されたコンテンツの定期表示や、インタラクティブな地図や、 2D/3D グラフィックのアニメーションや、ビデオジュークボックスのスクロールなど — たぶん JavaScript が組み込まれています。
